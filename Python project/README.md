# Student Result Management System

A Python-based console application designed to manage student academic records.

## Features
- Add students
- Add courses and scores
- Automatic GPA calculation
- Persistent data storage using JSON
- Menu-driven terminal interface

## Technologies Used
- Python
- JSON file storage

## Problem It Solves
Many schools and tutorial centers still calculate results manually, which leads to errors and data loss.  
This system automates result recording and GPA calculation while keeping records safe.

## How to Run
1. Make sure Python is installed
2. Open terminal in the project folder
3. Run:
   ```bash
   python student_result_system.py
